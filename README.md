# Notify Backend for inhouse deployment


Requirtements: 
  * docker-engine: 17.04.0+
  * docker-compose version 3.2

To compile and start:
  docker-compose up -d
  
To kill and clear:
  docker-compose down
  
To stop:
  docker-compose stop
  
To start again:
  docker-compose start
  
View logs:
  docker logs notify-backend
